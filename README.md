# ZMK Config

This repo stores keymaps for my keyboards running on ZMK.

## Available Keymaps / Branches

### Dumbpad

- `dumbpad` Dumbpad running on a nice!nano v2
  - `default` default keymap
  - `oled_top` keymap for use with oled and (left) encoder rotated to the top

### Eternal Keypad

- `eternal_keypad` Eternal Keypad running on a nice!nano v2
  - `default` default keymap
  - `lefty` keymap for left handed gamers
